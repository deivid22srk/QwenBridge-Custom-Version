## Shipped in code

### Responses API
- `reasoning.effort`: accepts Codex `xhigh` / `none` / `minimal` / `max` → normalize low|medium|high (`effort.ts`)
- `low` effort → `*-no-thinking` model; high/xhigh keep thinking
- SSE: `event:` + `data:` + `sequence_number`
- `last_response_id` on response objects
- `session_id` / metadata → chat thread key
- Multimodal: `input_image` / `input_file` → chat `image_url` / `file_url`
- Built-in tools drop **logged** (14→10 is not a hard cap)
- Error envelope OpenAI-shaped: `{ error: { message, type, param, code } }`
- **SQLite durable** `previous_response_id` store (`responses_store` table)
